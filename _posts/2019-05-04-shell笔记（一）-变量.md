### 特殊变量

- $0=脚本名称
- $1-$9=脚本执行的参数1-9
- $?=脚本的返回值（最后命令的退出状态，0表示没有错误）
- $$=当前脚本的进程id
- $!=当前进程的最后一个id
- $#=脚本的输入参数个数
- $@=输入参数内容（列表形式）
- $*=输入参数内容（整体作为一个单字符串）

字符串中使用要加转义

\\$*

### 变量

定义时不加$，变量名：字母数字下划线（数字不能最先）

```shell
your_name="zyx"
```

*变量和等号中间不能有空格*

#### 语句赋值

```shell
for file in 'ls /etc'

for file in $(ls /etc)

```

#### 使用变量

$变量

```shell
your_name="2333"
echo $your_name
echo ${your_name}
#花括号可选，用于区分边界：考虑$javascript和${java}script
#建议全都加{}
```

可以再次赋值

```shell
your_name="tom"
echo $your_name
your_name="alibaba"
echo $your_name
#不是${your_name}=“tom"
```



#### 只读变量、删除变量

readonly命令

```shell
name="asdasd"
readonly name
#此时再给name赋值
#name="aaaaaa"
#会报错
```

unset命令

```shell
name="a"
unset name
echo $name
#删除后不能使用，但是不能删除只读变量
```
